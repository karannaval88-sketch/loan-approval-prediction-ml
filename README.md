# 💰 Loan Approval Prediction (Machine Learning + Power BI)

## 📌 Overview
This project predicts whether a loan application will be approved or not based on applicant details using machine learning models.  
Additionally, a Power BI dashboard is created to visualize key insights from the dataset.

---

## 📊 Dataset
The dataset includes:
- Gender
- Marital Status
- Education
- Applicant Income
- Loan Amount
- Credit History
- Property Area

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn
- **Power BI (for dashboard visualization)**

---

## 🚀 Workflow
1. Data Cleaning and Handling Missing Values  
2. Encoding Categorical Variables  
3. Feature Scaling  
4. Model Training (Logistic Regression, Decision Tree)  
5. Model Evaluation  
6. Data Visualization using Power BI  

---

## 📈 Results
- Built a classification model to predict loan approval  
- Evaluated using:
  - Accuracy Score  
  - Confusion Matrix  
- Identified key factors affecting loan approval such as income, credit history, and loan amount  

---

## 📊 Power BI Dashboard
A Power BI dashboard was created to provide interactive insights:

### Key Visuals:
